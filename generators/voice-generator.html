How to Generate Voice Files:

  1. Find Your Perfect Voice:
    • Select a voice from the dropdown
    • Adjust pitch, rate, and volume sliders
    • Click "Preview '1'" to test the sound
    • Click "Play All Numbers" to hear the full sequence
  2. Option A - Manual Recording (Free):
    • Use audio recording software (e.g., Audacity, QuickTime, Windows Voice Recorder)
    • Start recording and click "Play All Numbers"
    • Split the recording into individual files (1.wav, 2.wav, etc.)
    • Save as WAV or MP3 format
  3. Option B - TTS Services (Best Quality):
    • Google Cloud TTS: cloud.google.com/text-to-speech
    • Amazon Polly: aws.amazon.com/polly
    • ElevenLabs: elevenlabs.io (Very natural voices)
    • Generate files "1", "2", "3", etc. up to your max number
    • Download as WAV files for best quality
  4. Use the Files:
    • Name files as: 1.wav, 2.wav, 3.wav, etc.
    • Upload to your metronome project
    • Update the metronome code to load and play voice samples

Why can't this tool generate files directly? Browser security restrictions prevent direct recording of Speech Synthesis API output. However, this tool helps you find the perfect voice settings before recording!
